# Gestor de Correos - Backend

Backend de Node.js + Express para el sistema de gestión de correos de la agroexportadora.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Producción

```bash
npm start
```

## Estructura

- `src/config/` - Configuración de base de datos
- `src/controllers/` - Lógica de negocio
- `src/routes/` - Rutas de la API
- `src/services/` - Servicios auxiliares
- `src/utils/` - Utilidades

## Variables de Entorno

Ver `.env.example`
